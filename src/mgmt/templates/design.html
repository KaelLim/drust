drust design system

Cozy-dark palette · apricot accent (hue 50) · Newsreader / Inter / JetBrains Mono. All tokens defined in _styles.html. New components are additive — existing pages re-theme automatically through token swap.

Color tokens

Surfaces, ink, and status families. All values are OKLCH, so chroma and lightness move predictably under hue shift. Background hue is ~50 (warm apricot family); accents hold their own hues.

Command palette

Press ⌘ K (or Ctrl K) to open. Anchors to top-center; backdrop blur; arrow keys + enter to navigate.

Modal

Used for confirmations and "secret-once" reveals. Inline preview below — wiring lives in _modal.html.

Rotate service token?

The current service token will be invalidated immediately. Any clients using it must be reconfigured with the new value. This action cannot be undone.
